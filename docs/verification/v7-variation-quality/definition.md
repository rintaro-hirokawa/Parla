# V7: 類題生成の品質と多様性

## 問い

SRSレビュー用に、同一の学習項目に対して毎回異なる文脈で自然な類題を生成し続けられるか?

## 背景

SRSレビュー（ブロック1）では、過去にストックした学習項目を「別の文脈で」練習する。同じ文脈で繰り返すと、パターンではなく特定の文を暗記してしまい、「文脈非依存の定着」（06-srs-mastery.md）が達成できない。

LLM #3 は対象学習項目 + ユーザー登録済みソースの内容を使って類題を生成する。

不確実な点:
- 同じ学習項目に対して5回、10回と生成を繰り返した時に、過去の類題と重複しない多様な文を作れるか
- ソースの内容を活かしつつ自然��文脈で学習項目を使った例文を生成��きるか
- 学習項目の使い方が模範として正しいか（不自然な文に無理やり埋め込んでいないか）

## 検証��法

### テストセットの準備

以下の学習項目を用意:
- "be responsible for ~ing"（構文）
- "environmental impact"（コロケーション）
- "more X than Y"（文法: 比較級）
- "take into account"（表現）

### 検証手順

1. 各学習項目に対して、異なるソース内容を文脈として5回ずつ類題を生成
2. さらに同じソースで5回生成（計10回）
3. 出力を評価

### 評価観点

| 観点 | 確認内容 |
|------|---------|
| 多様性 | 10回の生成で内容が重複していないか |
| 文脈の自然さ | ソース内容を活かした自然な文脈か。無理やり感がないか |
| 項目の正確な使用 | 学習項目が文法的・意味的に正しく使われているか |
| CEFRレベル適合 | 指定したCEFRレベルに合った難易度か |
| 日本語プロンプト | 日本語のお題として自然か |

## 合格基準

- 10回の生成で明確な重複がない（同じ文構造の繰り返しが20%未満）
- 学習項目の使い方が正しい率: 90%以上
- 文脈が自然な率: 80%以上

## フォールバック

- 過去の類題履歴をプロンプトに含めて重複回避を強制する（プロンプト長が増加するトレードオフ）
- 類題のテンプレートバリエーション（疑問文、否定文、受動態等）を指定して多様性を担保する
- ソース文脈の使用を諦め、汎用的な文脈で生成する
