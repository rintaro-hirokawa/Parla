# 画面設計・遷移フロー

注意: 本ドキュメントのUI例（ASCIIモックアップ）は、機能要件を説明するための参考資料である。実際のUI実装時には、プラットフォーム（デスクトップ/モバイル）および利用するUIライブラリ・フレームワークの特性を考慮して設計すること。

---

## 全画面一覧

### A. 共通レイアウト

- メインタブ4つ: 今日の学習 / 学習項目 / 学習履歴 / 設定
- セッション中はタブバーを非表示とし、学習フローに没入させる

### B. 初回セットアップ (SCREEN-B)

- CEFRレベル選択（A1〜C2）
  - 各レベルに平易な日本語説明を付与する（例: A1「簡単な自己紹介や日常のやりとりができるレベル」）
- 英語バリエーション選択（American / British / Australian / Canadian / Indian）

### C1. 今日の学習タブ (SCREEN-C1)

- 本日のセッションメニューを表示する
- 各ブロックの内容と推定時間を一覧表示する
- 「学習開始」ボタンを配置する
- メニュー内容はユーザーが変更できない（変更は前日のメニュー確定時に実施する設計）

### C2. 学習項目一覧タブ (SCREEN-C2)

- ストック済み学習項目のフィルタ付きリスト
- フィルタ条件: SRS段階、カテゴリ、ソース、習得状態

### C3. 学習項目詳細 (SCREEN-C3)

- 学習項目の成長ストーリー（初出時の発話 → 各復習での発話推移）
- WPM推移グラフ

### C4. 学習履歴タブ (SCREEN-C4)

- カレンダー表示（学習実施日をマーク）
- 累計WPM推移グラフ
- 日別の学習サマリー

### C5. 設定タブ (SCREEN-C5)

- CEFRレベル変更（システムからの提案がある場合は提案内容を表示）
- 英語バリエーション変更
- ソース管理画面へのリンク
- 発音記号表示 ON/OFF 切替

### D1. ソース登録 (SCREEN-D1)

- テキスト入力ボックス（100〜50,000文字）
- タイトル入力欄
- 適用されるCEFRレベルの表示（現在の個人設定値を自動読み込み）

### D2. ソース一覧 (SCREEN-D2)

- フィルタ付きリスト（ステータス、CEFR等）
- 各ソースの進捗をプログレスバーで表示する
- 各ソースの学習完了状態を表示する

### E1. マイクチェック (SCREEN-E1)

- リアルタイム波形表示
- レベルメーター
- マイク選択ドロップダウン
- 入力レベルが低すぎる場合は警告を表示する

### E2. ブロック1復習 (SCREEN-E2)

- 学習項目名の表示（SRS段階に応じてカテゴリレベルの示唆のみの場合あり）
- 日本語お題
- タイマー
- リアルタイム波形表示
- ヒント表示
- 正解時: 簡易フィードバック → 1.5秒後に自動遷移
- 不正解時: 模範解答表示 → ヒント → リトライ（最大3回）

### E3. フェーズA発話 (SCREEN-E3)

- パッセージの全センテンスを一覧表示する
- 現在のセンテンスをハイライトする
- タイマー
- リアルタイム波形表示
- ヒント表示（[07-hint-system.md](07-hint-system.md) 参照）
- ヒントボタン: ストック済み学習項目がセンテンスに関連する場合、ボタン押下で表示

### E4. フェーズBフィードバック (SCREEN-E4)

- ユーザーの発話再現テキスト
- 動的模範解答
- 学習項目リスト（自動ストック済み / 要確認 / 再出フラグ付き）
- 即時リトライ機能

### E5. 学習項目の任意修正シート (SCREEN-E5)

- フェーズBで自動ストックされた学習項目を、必要な時だけ編集・除外・手動追加するためのシート

### E6. フェーズC練習ワークスペース (SCREEN-E6)

- 同一画面上でリスニング / オーバーラッピング / 本番発話を切り替えられる
- TTS再生コントロール
- 英文テキスト表示
- 日本語お題表示（本番発話モード時は主表示）
- 学習項目部分のハイライト
- 発音記号表示（各単語の下に小さく表示、設定でON/OFF切替可能）
- 再生速度調整
- リアルタイム波形表示
- オーバーラッピング時は再生位置のリアルタイムハイライト
- 本番発話の結果表示（合格 / 不合格 / 差分要約）
- 推奨アクションは表示してよいが、練習順序や回数は固定しない

### E9. パッセージ完了サマリー (SCREEN-E9)

- パッセージの学習結果サマリー（パッセージ学習完了 / 通し発話達成の有無）
- ストックした学習項目の一覧
- WPM結果

### F1. セッション終了サマリー (SCREEN-F1)

- 今日の成果（学習パッセージ数、新規学習項目数、復習正答率等）
- WPM推移グラフ

### F2. 明日のメニュー確定 (SCREEN-F2)

- システムがルールに基づき構成した明日のメニューを表示する
- 各ブロックの内容と推定時間
- 素材の変更が可能（進行中の素材一覧 / 新しいソースの追加）
- 「このメニューで確定」ボタン

## セッション中の共通ヘッダー

学習中は画面上部にステータスバーを常時表示する。

| 表示項目 | 説明 |
|---------|------|
| 現在のブロック名と進捗 | 例: 「ブロック1 復習 (8/20)」 |
| 経過時間 | セッション開始からの累計時間 |
| 累計発話ワード数 | セッション内で発話したワード数の累計 |
| 平均WPM | セッション内の平均発話速度 |

---

## 録音に関する共通UI

- 録音中はレベルメーターおよび波形をリアルタイムで表示する
- 録音状態が視覚的に確認できることで「録音できている感」を演出する
- 発音記号: 英文テキスト表示時、各単語の下に小さく発音記号を表示可能（設定でON/OFF切替）

## 画面遷移フロー

```
[C1 今日の学習] --「学習開始」--> [E1 マイクチェック]
    |
    v
[E1 マイクチェック] --「開始」--> [E2 ブロック1 復習]
    |
    v
[E2 ブロック1 復習] -- 全問完了 --> [ブロック2 開始]
    |
    v
=== ブロック2 新規素材学習 ===
    |  パッセージごとに以下を繰り返す:
    |    [E3 フェーズA] --> [E4 フェーズB]
    |      -- Phase Cを使わない --> [E9 パッセージ完了サマリー]
    |      -- Phase Cを使う --> [E6 フェーズC練習ワークスペース] --> [E9 パッセージ完了サマリー]
    |
    v
[E2 ブロック3 定着] -- 全問完了 --> [F1 セッション終了サマリー]
    |
    v
[F1 セッション終了サマリー] --> [F2 明日のメニュー確定]
    |
    v
[F2 明日のメニュー確定] --「確定」--> [C1 今日の学習（翌日）]
```

### フェーズCワークスペース内の利用イメージ

```
[E6 フェーズC練習ワークスペース] -- リスニング / オーバーラッピング / 本番発話を任意に実行 --> [E9 パッセージ完了サマリー]
```

### フェーズBの即時リトライフロー

```
[E4 フィードバック] -- リトライ --> [発話] --> [判定]
                     -- 正解 --> 次のセンテンスまたは自動反映結果の確認
                     -- 不正解 --> 再リトライ（最大3回）
```
