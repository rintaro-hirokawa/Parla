# モチベーション・進捗可視化

---

## WPMをメイン成長指標とする

- 「1分間に何語話せるか」がスピーキング学習の最も直感的なゴール指標である
- 正答かつ時間内に発話できた場合のみWPMを記録する（不正解は除外）
- CEFRレベル別のターゲットWPMと比較して進捗を可視化する

### WPM計測方法

```
WPM = (模範解答のワード数) / (ユーザーの発話所要時間[分])
```

---

## セッション終了サマリー

セッション終了時に以下の情報を表示する。

- 学習時間
- 発話ワード数
- 完了パッセージ数
- 新たにストックした学習項目一覧
- 復習の正答率
- 習得済みに昇格した項目（少し目立つ演出あり）
- WPMの推移（先週比）
- ターゲットWPMまでの距離
- 累計情報
  - 総習得項目数
  - 学習中項目数
  - 連続学習日数

---

## リアルタイム表示（学習中）

画面のバー（トップまたはボトム）に以下をリアルタイム表示する。

| 表示項目 | 説明 |
|---------|------|
| 現在のブロック名と進捗 | 例: ブロック2 新規学習 (3/5) |
| 経過時間 | セッション開始からの経過時間 |
| 累計発話ワード数 | リアルタイム加算 |
| 現在の平均WPM | リアルタイム更新 |

---

## 学習履歴タブ

### カレンダー表示

- 日ごとの学習有無を表示する
- 日付クリックでその日の学習サマリーを表示する

### 推移グラフ

以下のグラフを切り替え（ドロップダウン）で表示する。

- 発話ワード数の推移
- WPM改善推移
- 習得済み学習項目数の累積推移

### カテゴリ別の習得状況

大分類ごとの習得済み項目数を表示する（例: 文法: 20個、語彙: 15個、コロケーション: 12個）。

---

## 学習項目詳細画面の成長ストーリー

- 初出ソース
- 全練習履歴（日時、正解/不正解、WPM、タイマー消費率）
- WPM推移グラフ

---

## 入れないもの

以下の機能は意図的に含めない。

- 他ユーザーとのランキング
- ゲーミフィケーション過剰（バッジ、レベルアップ等）
- 不確実な予測（「N日で目標達成」等）
