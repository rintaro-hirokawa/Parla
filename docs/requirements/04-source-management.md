# ソース登録・素材管理

ユーザーが学習素材（ソース）を登録し、AIがパッセージを生成・管理する仕組みを定義する。

---

## 入力方式

- テキストボックスにテキストを貼り付ける（YouTube動画の要約、記事、ニュース等なんでもOK）
- タイトルは任意（未入力ならAIが内容から自動生成）
- 将来的にYouTube URL対応（URLを検知 -> 自動テキスト抽出 -> テキストボックスに展開）

---

## バリデーション

| 項目 | ルール |
|------|--------|
| 最小文字数 | 100文字以上 |
| 最大文字数 | 50,000文字以下 |
| 言語 | 日本語・英語・混在すべてOK |
| 内容チェック | パッセージ生成に失敗した場合のみエラー表示。事前の内容検閲はしない |
| 重複チェック | 行わない |

---

## ソースのCEFRレベル

- ソース登録時に個人設定のCEFRレベルを読み込み、そのソースに紐付けて固定する
- 途中で個人設定のCEFRレベルを変更しても、既存ソースには影響しない
- 画面上に適用中のCEFRレベルを小さく表示する

---

## ライフサイクル

```
登録 -> AI生成中 -> 未着手 -> 学習中(N/Mパッセージ) -> 完了 -> アーカイブ
```

- 生成失敗時: ステータス「生成失敗」 -> ユーザーに通知し、再登録を促す

---

## パッセージ生成

- LLMが最適なパッセージ数・分量を決定する（固定ではない）
- パッセージ生成時に事前模範解答+各センテンスのヒント（[07-hint-system.md](07-hint-system.md) 参照）も同時生成する
- 生成はソース登録時に非同期で実行する。完了までステータスは「AI生成中」とする

---

## 学習スタイル

- すべてのソースはL1（和文英訳型）で学習する
- L2（要点指示型）、L4（英語プロンプト型）での再学習はMVP後に検討する

---

## 管理UI

- テキスト検索（タイトル、内容）
- タグ（AIが内容から自動付与+ユーザー編集可能。例: "DIY", "ビジネス", "技術"）
- ステータスフィルタ（未着手 / 学習中 / 完了）
- ソート（最終学習日順がデフォルト、登録日順も可）
- フォルダ分類は不要（半年で100〜200件程度の見込み）
- プログレスバーで進捗表示（N/Mパッセージ）
- 各ソースにストック済み学習項目数を表示する
