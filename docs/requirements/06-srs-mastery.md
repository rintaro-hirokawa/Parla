# SRS・定着判定

---

## 学習項目SRS

- L1スタイル（和文英訳の単文）で復習する
- 文脈を変える: 毎回異なるソースの内容をベースにした文で出題する

---

## SRSパラメータ

### 間隔テーブル

| 段階 | 間隔 |
|------|------|
| 0 | 同日 |
| 1 | 1日 |
| 2 | 3日 |
| 3 | 7日 |
| 4 | 14日 |
| 5 | 30日 |
| 6 | 60日 |

### 不正解時の処理

- 間隔を1段階戻す

### 反応速度による判定

- タイマー80%以上消費で正解した場合、間隔を進めない
- 理由: 考えて出せる状態は定着とは判断しない

### 信頼度乗数

次回間隔の計算式:

```
next_interval = current_interval * ease_factor * confidence
```

ヒント使用状況に応じた信頼度乗数（詳細は [07-hint-system.md](07-hint-system.md) を参照）:

| ヒント使用 | 信頼度乗数 |
|-----------|-----------|
| ヒントなし | 1.0 |
| ヒント1 | 0.7 |
| ヒント2 | 0.4 |

### 文脈変化による定着判定

- 3回以上異なる文脈で正解した場合、文脈非依存の定着と判断する

### フェードアウト

- 完全卒業ではなく、出題頻度を漸減させる

---

## 類題生成ルール

SRS復習時の類題生成には以下のルールを適用する。

- ユーザーが登録しているソースの内容を使った文にする
- 表現をピックした元のソースとは異なるソースをベースにして文脈を変える
- 前回とほぼ同じ文にならないよう配慮する（過去の類題履歴を参照）
- 親和性の高い組み合わせで文脈変化させつつ、学習項目は同じものを使う
