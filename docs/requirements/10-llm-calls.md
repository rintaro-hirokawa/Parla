# LLMコール一覧

本ドキュメントでは、システムが使用するLLMコールの全量を定義する。

---

## コール一覧

### #1 パッセージ生成（事前・非同期）

| 項目 | 内容 |
|------|------|
| タイミング | ソース登録時（非同期） |
| Input | ソーステキスト、CEFRレベル、英語バリエーション、パッセージタイプ |
| Output | パッセージ群（日本語お題、事前模範英文、タイプ）、各センテンスのヒント（[07-hint-system.md](07-hint-system.md) 参照） |

---

### ~~#2 セッション構成~~ （廃止: 決定論的ルールベースに移行）

セッション構成は決定論的ルールで実現する（[03-session-structure.md](03-session-structure.md) 参照）。LLMコールは不要。

---

### #3 類題生成

| 項目 | 内容 |
|------|------|
| タイミング | ブロック1用=前日生成、ブロック3用=パッセージ完了時バックグラウンド |
| Input | 対象学習項目、CEFRレベル、英語バリエーション、文脈用ソース内容、過去の類題履歴 |
| Output | 類題（日本語お題、模範英文） |

---

### #4 フィードバック生成（センテンスごと）

| 項目 | 内容 |
|------|------|
| タイミング | フェーズA中バックグラウンド並列 |
| Input | 発話音声ファイル、日本語お題、事前模範英文、CEFRレベル、英語バリエーション、既存ストック済み学習項目リスト（再出検知用）、パッセージ文脈 |
| Output | 発話再現、動的模範解答、学習項目リスト（新規/再出フラグ、confidence、default_action: auto_stock / review_later） |

---

### #5 即時リトライ判定

| 項目 | 内容 |
|------|------|
| タイミング | フェーズB中リアルタイム |
| Input | リトライ発話音声、動的模範解答、日本語お題、CEFRレベル |
| Output | 正解/不正解、一言コメント（出力トークン最小限） |

---

---

### ~~#8 本番発話の内容評価~~ （LLMコールから Azure Speech Service に移行）

Azure Speech Service のストリーミング Pronunciation Assessment + difflib による判定に移行した（[技術検証 V11](../verification/v11-full-passage-evaluation/definition.md)）。LLMコールは不要。

| 項目 | 内容 |
|------|------|
| タイミング | 本番発話中（ストリーミング）〜発話直後 |
| 方式 | Azure Pronunciation Assessment（reference_text=動的模範解答）+ difflib で Omission/差分検出 |
| Input | 発話音声（PushAudioInputStream でリアルタイム送信）、動的模範解答 |
| Output | 合否判定、文ごとの一致度（similarity）、Omission 箇所 |
| レイテンシ | 発話終了後 ~1.2秒 |

---

### ~~#11 素材再学習の動的再構成~~ （MVP後に検討）

L2/L4での再学習機能とともにMVP後に検討する。

---

### #12 ループ・疲労管理

| 項目 | 内容 |
|------|------|
| タイミング | フェーズCで本番発話の不合格や練習停滞が続いた時 |
| Input | ループ回数、経過時間、パフォーマンス推移、学習項目難易度 |
| Output | ユーザーへの短いメッセージ、推奨アクション |

---

## 事前生成可能 vs リアルタイム必須

| 分類 | コール |
|------|--------|
| 事前生成可能 | #1 パッセージ生成、#3 類題生成 |
| リアルタイム必須（音声入力が必要） | #4 フィードバック生成、#5 即時リトライ判定 |
| Azure Speech Service（LLM不要） | ~~#8~~ 内容評価（ストリーミング Pronunciation Assessment） |
| セッション中の状態依存 | #12 ループ・疲労管理 |

## 共通インプット

### ほぼ全コールで必要

- CEFRレベル
- 英語バリエーション

### 多くのコールで必要

- 既存ストック済み学習項目リスト
- ソース一覧と進捗
